golden-ratio.vim

Automatic Window Re-sizing

Intro

This plugin resizes automatically the windows of your editor to the size specified in the Golden Ratio. The window that has the main focus will have the perfect size for editing, while the ones that are not being actively edited will be re-sized to a smaller size that doesn't get in the way, yet at the same time will be readable enough to know it's content.

Options

g:loaded_golden_ratio
Use this option to disable the plugin completely. 

let g:loaded_golden_ratio = 1
g:golden_ratio_autocommand
Use this option to disable automatic re-sizing when moving between windows 

let g:golden_ratio_autocommand = 0

Commands

:GoldenRatioToggle
This will switch on and off the automatic window re-sizing
:GoldenRatioResize
When the automatic window re-sizing is off and this command is called, the active window will be re-sized to the Golden Ratio size
